## Bookshelf
A personal project to track reading across multiple languages and keep short reading journals.

### Live Demo
https://my-bookshelf-hazel.vercel.app

### Tech Stack
Frontend
 * React
 * TailwindCSS
 * Vercel
Backend
 * Spring Boot
 * JPA / MyBatis
 * JWT Authentication
 * Render
Database
 * PostgreSQL
 * Supabase
